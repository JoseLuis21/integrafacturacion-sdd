# Recommended MCPs

MCPs (Model Context Protocol tools) help AI interact with the development environment.

These are optional but highly useful for AI-assisted workflows.

---

# Filesystem MCP

Purpose:

Allow AI to:

- read project files
- edit files
- inspect repository structure

Useful for:

- implementing modules
- editing specs
- generating boilerplate

---

# GitHub MCP

Purpose:

Allow AI to:

- read repositories
- analyze pull requests
- review code changes

Useful for:

- code review
- repository navigation
- architecture validation

---

# Postgres MCP

Purpose:

Allow AI to inspect database schemas.

Useful for:

- validating migrations
- writing queries
- understanding relations

---

# HTTP MCP (Optional)

Purpose:

Allow AI to test endpoints.

Useful for:

- validating API responses
- testing authentication flows
- debugging APIs

---

# When to Use MCPs

MCPs are useful when AI needs to:

- inspect the project
- modify files
- validate runtime behavior

They are **not required** for simple prompt-based workflows but improve automation significantly.
